# 二分查找

## 二分查找的前提

1. 目标函数单调性（单调递增或单调递减）
2. 存在上下界（bounded）
3. 能够通过索引访问（index accessible）

## 代码模版

```python
left, right = 0, len(array) - 1
while left <= right:
  mid = (right - left) / 2
  if (array[mid] == target):
    # find the target
    break or return result
  elif (array[mid] < target):
    left = mid + 1
  else
    right = mid - 1
```

```js
function find(array = [], target) {
  let left = 0;
  let right = array.length - 1;
  while (left <= left) {
    const mid = Math.floor((right - left) / 2);
    if (array[mid] === target) {
      // find the target
      // break or return result
    } else if (array[mid] < target) {
      left = mid + 1;
    } else {
      right = right - 1;
    }
  }
}
```
